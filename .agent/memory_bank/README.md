# Project Memory Bank

This directory contains the "Memory Bank" for the Portfolio project. It serves as a central knowledge base for the AI agent and the developer to track the project's state, structure, and context.

## Files

- **[project_brief.md](./project_brief.md)**: High-level overview of the project, goals, and tech stack.
- **[structure.md](./structure.md)**: Detailed documentation of the file structure and component architecture.
- **[scripts.md](./scripts.md)**: List of available NPM scripts and their purposes.
- **[conventions.md](./conventions.md)**: Coding standards and best practices for this project.
- **[active_context.md](./active_context.md)**: A living document tracking the current task, recent changes, and next steps. **Update this file at the start and end of every session.**

## Usage

1.  **Start of Session**: Read `active_context.md` to understand where the last session left off.
2.  **During Work**: Update `active_context.md` with notes, decisions, and progress.
3.  **End of Session**: Update `active_context.md` with a summary of changes and the next steps.
4.  **Architecture Changes**: If you add new directories or major components, update `structure.md`.
5.  **New Scripts**: If you add new `package.json` scripts, update `scripts.md`.
