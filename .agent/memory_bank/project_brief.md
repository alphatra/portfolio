# Project Brief: Portfolio App

## Overview
This is a personal portfolio application built with Astro, React, and Tailwind CSS. It features a modern, dynamic design with animations and interactive elements. The project serves as a showcase for the user's work, skills, and blog posts.

## Core Technologies
- **Framework**: [Astro](https://astro.build/) (v5.x)
- **UI Library**: [React](https://react.dev/) (v19.x)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) (v4.x)
- **Animation**: 
  - [Framer Motion](https://www.framer.com/motion/)
  - [Motion](https://motion.dev/)
  - [tsparticles](https://particles.js.org/)
  - [Cobe](https://github.com/shuding/cobe) (Globe animation)
- **Deployment**: Vercel

## Key Features
- **Responsive Design**: Mobile-first approach using Tailwind CSS.
- **Dark/Light Mode**: Themed UI with toggles.
- **Internationalization (i18n)**: Support for multiple languages (implied by `src/i18n`).
- **Content Management**: Likely uses Astro Content Collections for blog posts and projects.
- **Interactive Components**:
  - Bento Grid layout for projects/skills.
  - 3D Globe visualization.
  - Spotify/Last.fm integrations.
  - GitHub activity feed.
  - Guestbook.

## Goals
- Showcase professional work and research.
- Provide a blog for sharing thoughts.
- Demonstrate technical skills through the portfolio's own implementation.
